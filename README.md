RED Boilerplate
==========================

RED Interactive's custom Robyn boilerplate.

### Requirements

- [`robyn`](http://github.com/ff0000/robyn) `>= 1.0.0`

### Install the Robyn CLI

```bash
npm install robyn -g
robyn add rbp git://github.com/ff0000/red-boilerplate.git
```

How to use
==========

### Creating a project

```bash
robyn init rbp path/to/folder
```
